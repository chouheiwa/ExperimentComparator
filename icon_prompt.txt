# 数据对比工具图标生成提示词

## 主要提示词 (Primary Prompt)

Create a modern, professional desktop application icon for a data comparison tool. The icon should feature:

**核心元素 (Core Elements):**
- Two image frames side by side representing data comparison
- A subtle arrow or comparison symbol between them
- Clean, geometric design with rounded corners
- Modern flat design style with subtle gradients

**颜色方案 (Color Scheme):**
- Primary: Deep blue (#2563eb) or tech blue
- Secondary: White and light gray accents
- Accent: Green for "GT" (Ground Truth) elements
- Background: Subtle gradient from blue to darker blue

**设计风格 (Design Style):**
- Minimalist and professional
- 512x512 pixels, suitable for desktop applications
- Clean lines, no excessive detail
- Modern tech aesthetic
- Suitable for both light and dark themes

**具体描述 (Specific Description):**
Desktop application icon, two rectangular frames representing images with a comparison arrow between them, blue gradient background, white/gray accents, modern flat design, professional tech style, clean geometry, 512x512 resolution, suitable for macOS/Windows desktop

## 技术要求 (Technical Requirements)

- **尺寸**: 512x512 pixels (正方形)
- **格式**: PNG with transparent background preferred
- **分辨率**: High resolution for desktop use
- **兼容性**: Works on both light and dark backgrounds
- **风格**: Modern, professional, suitable for business software

---

## 避免的元素 (Elements to Avoid)

- 过于复杂的细节 (too many details)
- 文字内容 (text content)
- 过于花哨的装饰 (overly decorative elements)
- 暗色背景 (dark backgrounds)
- 过于卡通的风格 (cartoon style)

## 使用建议 (Usage Tips)

1. 将主要提示词复制到AI图像生成工具

3. 可以尝试不同的版本获得最佳效果
